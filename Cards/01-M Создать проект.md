# M-1: Создать проект
#everyone #tech 
1) Добавляем проект с именем config.

``` bash
django-admin startproject config
>>>
config/ 
	manage.py 
	config/ 
		__init__.py 
		settings.py 
		urls.py 
		asgi.py 
		wsgi.py
```
2) Внешнюю папку переименовываем в app

``` bash
app/ 
	manage.py 
	config/ 
		__init__.py 
		settings.py 
		urls.py 
		asgi.py 
		wsgi.py
```
